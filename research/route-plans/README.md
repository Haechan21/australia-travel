# 로드트립 루트 후보 5개 안 — 종합 비교 (v2)

> **작성일**: 2026-03-15
> **목적**: 시드니공항 출발/반납 (5/24~5/29, 6일) 로드트립 루트 후보를 비교하여 최적 일정 선정
> **평가자**: 효율 전략가 김도현(A) / 감성 탐험가 이서아(B) / 현실주의 비평가 박지우(C)
> **기반**: [TRAVELER_PROFILE.md](../../docs/TRAVELER_PROFILE.md) 여행자 프로필 반영

## v1 → v2 주요 변경점

여행자 프로필 반영으로 전제 조건이 크게 바뀜:
- **첫날 900km 직행 가능** → GC 직행이 더 이상 페널티가 아님
- **매일 350~400km OK** → 중거리 이동일도 관광일로 활용 가능
- **야간 2시간 OK** → 일몰 후 이동에 여유
- **은하수 촬영 희망** → 다크스카이 장소 필수 포함
- **뷰 숙소 투자 의향** → 숙소 자체가 하이라이트 가능
- **좌측통행 경험 있음** → 첫날 해안도로도 가능

## 각 안 한줄 요약

### 4조: 스타라이트 루프 (88.0점) — [상세](4조_자유컨셉.md)
**시드니 → GC(900km) → 스프링브룩 은하수 → 바이런 → 콥스/도리고 → 포트스테판스 → 블루마운틴(뷰숙소+별밤) → 그랜드퍼시픽 → 시드니**
- S등급 6곳 올클리어 + 은하수 2회 기회(5/25 스프링브룩, 5/28 블루마운틴)
- 블루마운틴 뷰 숙소 + 그랜드퍼시픽 피날레. 총 ~2,607km이나 Day 1에 882km 집중
- 감성파 91점(전체 최고). 리스크: Day 3~4 연속 이동, 스프링브룩 구름

### 3조: 해안+내륙 폭포 루프 (83.0점) — [상세](3조_북쪽중거리_내륙.md)
**시드니 → GC(직행) → 바이런 → 콥스하버 → Waterfall Way(도리고) → 포트맥쿼리 → 포트스테판스(은하수) → 시드니**
- S등급 4곳 + 빡센 날/여유 날 교차 패턴 완벽 구현
- 기존 D안의 Day 4 420km 문제를 230km로 개선
- 균형 잡힌 점수(80~87). 블루마운틴 없음이 약점

### 2조: 남쪽해안 그랜드루프 (83.0점) — [상세](2조_남쪽해안_블루마운틴.md)
**시드니 → 그랜드퍼시픽(S) → 키아마 → 블루마운틴(S, 2일) → 시드니(Vivid Sydney)**
- 최단 이동거리(~630km). 바다→산 서사. 블루마운틴 은하수 촬영
- Vivid Sydney(5/22~6/13) 활용 가능. 시드니 여유
- S등급 2곳만. 북쪽 해안/GC 완전 포기

### 1조: GC 직행 + 남하 해안 (77.3점) — [상세](1조_GC직행_남하해안.md)
**시드니 → GC(직행) → 바이런 → 콥스/Waterfall Way → 포트맥쿼리 → 포트스테판스 → 시드니**
- 기존 A안 개선. S등급 4곳. Day 2 GC 피로회복일 배치
- 포트스테판스 핑갈베이 은하수 촬영
- 블루마운틴/그랜드퍼시픽 없음. 3조와 유사하나 열대우림 덜 깊음

### 5조: 풀 스윙 (75.0점) — [상세](5조_야심작.md)
**시드니 → GC → 바이런 → 콥스/도리고 → 포트스테판스(샌드보딩) → 블루마운틴(일출) → 그랜드퍼시픽 → 시드니**
- S등급 5곳. 해안+내륙+사막+산 4가지 풍경. 도리고 은하수
- 가장 야심찬 루트. 감성파 85점. 현실파 68점으로 리스크 큼
- Day 6 새벽 출발 + 블루마운틴 반나절이 최대 약점
