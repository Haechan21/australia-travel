# 로드트립 루트 후보 5개 안 — 종합 비교

> **작성일**: 2026-03-13
> **목적**: 시드니공항 출발/반납 (5/24~5/29, 6일) 로드트립 루트 후보를 비교하여 최적 일정 선정
> **평가자**: 효율 전략가 김도현(A) / 감성 탐험가 이서아(B) / 현실주의 비평가 박지우(C)

## 각 안 한줄 요약

### A안: 북쪽 풀코스 (63.7점) — [상세](A안_북쪽풀코스.md)
**시드니 → 골드코스트/바이런베이(900km 북상) → 해안 남하 → 시드니**
- 가장 넓은 범위. S등급 4곳 커버. 대신 첫날 900km 야간 운전, 총 2,130km 과다 이동
- 효율파·현실파 모두 낮은 점수. "이동이 여행을 잡아먹는" 루트

### B안: 남쪽 해안 + 블루마운틴 (77.3점) — [상세](B안_남쪽해안_블루마운틴.md)
**시드니 → 그랜드퍼시픽 → 울릉공/키아마 → 저비스베이 → 블루마운틴 → 시드니**
- 최고 평균점. 바다→산 서사 구조. 이동거리 최소급(~750km)
- 감성파 82점 최고. 약점: S등급 2곳만, 북쪽 해안 경험 없음, 날씨 의존도

### C안: 중북부 집중 (75.0점) — [상세](C안_중북부집중.md)
**시드니 → 뉴캐슬 → 포트스테판스 → 센트럴코스트 → 블루마운틴 → 시드니**
- 최단 이동거리(~630km). 하루 평균 1.5시간 운전. 블루마운틴 2일 집중 탐방
- 효율파 78점으로 공동 1위. 약점: 골드코스트/바이런베이 완전 포기, 센트럴코스트 가성비 의문

### D안: 콥스하버 절충 (76.0점) — [상세](D안_콥스하버절충.md)
**시드니 → 포트맥쿼리 → 콥스하버 → 뉴캐슬/포트스테판스 → 시드니**
- GC/바이런 생략, 콥스하버까지만 북상. 해안+폭포+열대우림 다양성 확보
- 3인 점수 편차 가장 적음(74~78). 약점: Day 4 장거리, 바이런베이 포기

### E안: 남북 하이브리드 (73.3점) — [상세](E안_남북하이브리드.md)
**시드니 → 뉴캐슬(북) → 시드니 → 그랜드퍼시픽(남) → 블루마운틴 → 시드니**
- S등급 4곳 전부 포함. 시드니 허브앤스포크. 다양한 경험
- 감성파 80점. 약점: 시드니 반복 통과, 로드트립 서사 약함, 숙소 이동 빈번

## 페르소나별 추천

### 효율 전략가 김도현이 추천하는 안
> **C안 또는 D안** (둘 다 78점)
> "C안은 이동이 적어 관광 밀도가 높고, D안은 거리는 좀 더 있지만 경험의 폭이 넓다. A안은 절대 안 된다."

### 감성 탐험가 이서아가 추천하는 안
> **B안** (82점)
> "바다에서 산으로 이어지는 서사가 완벽하다. 그랜드퍼시픽의 해안 절벽과 블루마운틴의 깊은 계곡이 대비를 이루는 구성. 단, E안의 S등급 전부 커버도 매력적."

### 현실주의 비평가 박지우가 추천하는 안
> **D안** (76점)
> "가장 균형잡힌 안. 3인 점수 편차가 가장 적다는 건 리스크가 분산되어 있다는 뜻. B안도 좋지만 블루마운틴 2일 연속 맑은 날씨가 필요해서 변수가 크다."

## 결정 포인트

사용자가 어떤 안을 선택할지는 다음 질문에 대한 답에 달려 있다:

1. **바이런베이/골드코스트가 꼭 필요한가?**
   - 꼭 가야 한다 → A안 (유일한 선택지, 단 피로 각오)
   - 없어도 된다 → B/C/D/E안

2. **어떤 경험을 우선하는가?**
   - 해안 절벽 + 블루마운틴 자연 → **B안**
   - 열대우림 + 폭포 + 야생동물 다양성 → **D안**
   - 여유로운 일정 + 피로 최소화 → **C안**
   - S등급 전부 커버 + 다양한 맛보기 → **E안**

3. **하루 몇 시간 운전이 괜찮은가?**
   - 1~2시간 → C안, B안
   - 3~4시간 → D안, E안
   - 5시간+ → A안
