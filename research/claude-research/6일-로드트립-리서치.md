## 3. 날짜별 상세 활동 가이드

### Day 1 (일 5/24) — 시드니 → 골드코스트 (882km, 순수 이동일)

> **⚠️ 이 날은 10시간 비행 직후 882km 장거리 운전. 안전이 최우선.**
> 야간 운전 ~3시간 불가피 (일몰 17:00 이후 울굴가~골드코스트 구간).

**타임라인:**
| 시간 | 내용 |
|------|------|
| 09:00 | SIXT 렌터카 인수. **톨패스(E-Tag) 여부 확인 필수**. 좌측통행 10분 적응 |
| ~11:15 | 헤더브레이 도착 (180km). **90분 휴식** — 점심 + 스트레칭 + 카페인 섭취 |
| 12:45 | 헤더브레이 출발 |
| ~15:30 | 포트맥쿼리 도착 (410km). **45분 휴식** — 주유(필수) + 간식 |
| 16:15 | 포트맥쿼리 출발 |
| ~17:50 | 울굴가/콥스하버 도착 (590km). **60분 저녁식사 + 휴식** |
| 18:50 | 울굴가 출발 — **야간 운전 구간 시작** |
| ~21:00 | **골드코스트 도착** (882km) |

**야간 운전 안전 수칙 (18:50~21:00, 콥스하버→골드코스트 300km):**
- 속도 95~100km/h로 감속 (제한 110km/h 구간에서도)
- **캥거루 위험 최고 구간**: 콥스하버~바이런베이 (150km, 풀숲/관목 지대)
- 하이빔 적극 활용 (대향차 없을 때), 동물 발견 시 급핸들 금지 (정면 유지)
- 45분마다 5분 정차 스트레칭
- 졸음 느끼면 **즉시** Safe Stop Area에서 20분 수면

**경유지 식사:**
- 헤더브레이: Highway Service Centre (맥도날드, 카페) — 빠르고 가벼운 식사
- 울굴가/콥스하버: 가볍게 저녁 (과식 금지 — 야간 운전 졸음 유발)

### Day 2 (월 5/25) — 골드코스트 종일 (유일한 GC 풀데이)

> 전날 882km 장거리 운전 피로 회복 + 골드코스트 핵심 체험.
> 스프링브룩 Plateau(주간)과 Natural Bridge(야간)는 **같은 날 비추** — 하루 종일 산길 운전.
> **해안가 여유 오전 + 동물원 오후 + 반딧불이 야간**이 최적 리듬.

**타임라인:**
| 시간 | 내용 |
|------|------|
| 09:00 | **Paddock Bakery** 아침 (Miami, 정원 카페, 크로와상+에그롤+커피) |
| 10:15~11:30 | **Burleigh Heads NP** 워크 (Oceanview Track 1.2km + Rainforest Circuit 2.3km) |
| 11:30~12:15 | **James Street** 카페거리 산책, 커피 |
| 12:30~14:30 | **Currumbin Wildlife Sanctuary** ($40~63, 코알라 안기, 캥거루, 로리킷) |
| 14:45~16:00 | **Harbour Town Outlets** + **Chemist Warehouse** 쇼핑 |
| 16:00~17:00 | 숙소 복귀, 휴식. **야간 투어용 따뜻한 옷 준비** (산 위 10°C 이하) |
| 17:00~17:45 | **이른 저녁** (kost/Jimmy Wah's, Rick Shores는 예약 필수) |
| 18:00 | **스프링브룩 Natural Bridge로 출발** (GC에서 ~1시간) |
| 19:00~20:00 | **Natural Bridge 반딧불이(Glow Worm) 관람** — 동굴 천장 수천 마리, 이 여행 하이라이트! |
| 21:00~21:30 | 골드코스트 복귀, 숙소 |

**반딧불이 투어:**
- 가이드 투어 $70~90/인 (픽업 포함, 야간 산길 운전 불필요 — **전날 피로 감안 가이드 투어 강력 추천**)
- 셀프 드라이브 시 무료 (야간 산길 운전 필요, 캥거루 주의)
- 일몰 후 어두울수록 잘 보임. 손전등 동굴 안에서 OFF

**저녁 식사 우선순위:**
1. **Rick Shores** (벌리, 팬아시안, $35~55) — 예약 60일 전부터, rickshores.com.au
2. **kost** (브로드비치, 한국-호주 퓨전, $25~45) — 한국인 입맛에 편안
3. **Jimmy Wah's** (벌리, 베트남 퓨전, $25~45) — 캐주얼, 가성비

**쇼핑 TIP:**
- Chemist Warehouse: 비타민(Swisse, Blackmores), 루카스 포포 크림, 선크림
- Harbour Town: UGG, Cotton On, Nike/Adidas 아울렛

### Day 4 (수 5/27) — Waterfall Way: 콥스하버 → 벨링겐 → 도리고NP → 포트맥쿼리

**포트맥쿼리 저녁:**
