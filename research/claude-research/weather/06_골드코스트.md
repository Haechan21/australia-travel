# 골드코스트 — 5월 하순 날씨 및 여행 환경

## 1. 기온 (Temperature)

5월 하순은 남반구 가을 말기(겨울 진입기)로, 선선하고 쾌적한 날씨가 이어진다.

| 구분 | 수치 |
|------|------|
| 평균 최고 기온 | 24–25 °C |
| 평균 최저 기온 | 13–14 °C |
| 평균 기온 (mean) | 약 20 °C |
| 일교차 | 약 11 °C |

### 한국 월별 비교

| 골드코스트 5월 하순 | 한국 비교 |
|---------------------|-----------|
| 최고 24–25 °C | 서울 5월 중순~하순 (최고 24–25 °C) 과 유사 |
| 최저 13–14 °C | 서울 10월 중순 (최저 12–14 °C) 과 유사 |
| 일교차 11 °C | 서울 가을(10월)과 비슷한 일교차 |

> **체감**: 한국의 5월 중순 낮과 10월 중순 아침·저녁을 합친 느낌. 낮에는 반팔도 가능하나 아침·저녁에는 긴소매나 가벼운 겉옷 필수.

---

## 2. 강수 (Rainfall)

5월은 골드코스트의 건기(Dry Season) 진입기로, 여름·초가을 대비 강수량이 크게 줄어든다.

| 구분 | 수치 |
|------|------|
| 월 강수량 | 69–100 mm |
| 비 오는 날 | 8–11일/월 |
| 하루 평균 강수 | 소량 (가벼운 소나기 위주) |

- 여름철(1–3월) 월 150–200 mm 대비 절반 이하
- 비가 와도 대부분 짧은 소나기(shower)로 하루 종일 내리는 경우 드묾
- 5월 하순은 특히 건조한 편으로, 야외 활동에 적합

---

## 3. 일조 / 일몰 (Sunrise & Sunset)

5월 하순은 동지(6월 21일) 약 한 달 전으로, 낮 길이가 점차 짧아지는 시기.

| 구분 | 수치 (약) |
|------|-----------|
| 일출 | 06:15–06:20 (AEST) |
| 일몰 | 17:05–17:10 (AEST) |
| 낮 길이 | 약 10시간 50분 |
| 월 일조 시간 | 약 259시간 (하루 평균 ~8.4시간) |

- 동지 최단일(10시간 24분)보다 약 25분 더 길다
- 하루 평균 일조 8시간 이상으로 맑은 날이 많다
- **일몰이 17시 즈음**이므로 오후 야외 활동은 16시까지 마무리 계획 권장
- UV 지수: 4–5 (보통) — 그래도 호주이므로 자외선 차단제 필수

---

## 4. 습도 / 바람 (Humidity & Wind)

| 구분 | 수치 |
|------|------|
| 평균 상대 습도 | 64–73% |
| 평균 풍속 | 16–21 km/h |
| 주요 풍향 | 남서~서풍 (SW–W) |

- 여름(80% 이상) 대비 습도가 낮아 쾌적
- 5월은 1년 중 가장 바람이 적은 달 중 하나
- 해안가에서는 바닷바람으로 체감 온도가 2–3°C 더 낮게 느껴질 수 있음
- 아침·저녁 해안 산책 시 바람막이 재킷 권장

---

## 5. 바다 수온 / 수영 (Sea Temperature & Swimming)

| 구분 | 수치 |
|------|------|
| 평균 수온 | 22–24 °C |
| 수온 범위 | 21–25 °C |
| 수영 쾌적도 | 보통~양호 |

### 해변별 수영 환경

| 해변 | 특징 |
|------|------|
| **Surfers Paradise** | 파도가 적당하고 패트롤 구간 넓음. 수온 22–24°C로 짧은 수영 가능 |
| **Burleigh Heads** | 남쪽 포인트로 파도 영향 적은 구간 있음. 수영·스노클링 가능 |

### 웻수트 필요 여부

| 활동 | 권장 장비 |
|------|-----------|
| 일반 수영 (30분 이내) | 없어도 가능 (추위에 민감하면 래시가드 권장) |
| 장시간 수영/스노클링 | 2mm 스프링수트 또는 래시가드 |
| 서핑 | 2mm 숏존 또는 3/2mm 풀수트 (바람 강한 날) |

> 22–24°C 수온은 쾌적한 편이나, 바람이 있는 날 물 밖에서 체감 온도가 떨어지므로 래시가드 이상 준비 권장.

## 7. 서핑 조건 (Surf Conditions)

5월은 골드코스트 서핑에 좋은 시기 중 하나. 남반구 가을~겨울 스웰이 본격화되는 시기.

### 주요 서핑 포인트

| 포인트 | 파도 유형 | 5월 컨디션 | 난이도 |
|--------|-----------|------------|--------|
| **Snapper Rocks** | 라이트핸드 포인트 브레이크 | 수퍼뱅크의 시작점. 2025년 사이클론 이후 샌드뱅크 변화 있음 — 현지 조건 확인 필요 | 중상급 |
| **Kirra** | 라이트핸드 배럴 | 3–5월 최적 시즌. 세계적 수준의 튜브 라이딩 가능 | 중상급 |
| **Burleigh Heads** | 라이트핸드 포인트 브레이크 | 남~남동 스웰에 안정적. WSL 대회 개최지 | 중급 |
| **Surfers Paradise** | 비치 브레이크 | 초·중급자 적합. 파도 크기 보통 | 초중급 |
| **Currumbin Alley** | 비치/포인트 | 소규모, 조용한 세션 가능 | 초중급 |

### 5월 서핑 환경

| 구분 | 내용 |
|------|------|
| 스웰 방향 | 남~남동(S–SE) 스웰이 주류 |
| 평균 파고 | 1.0–1.5 m (간헐적 2 m+) |
| 수온 | 22–24 °C |
| 바람 | 서풍(오프쇼어) 조건이 많아 파도 면이 깨끗 |
| 웻수트 | 2mm 숏존 또는 3/2mm 풀수트 |
| 크라우드 | 여름 대비 적음 — 로컬 서퍼 위주 |

> **참고**: 2025년 열대성 사이클론 알프레드로 인해 Snapper Rocks~Superbank 구간의 샌드뱅크가 크게 변화하였음. 2026년 5월 시점 복구 상황은 현지 서핑 리포트(Surfline, Swellnet) 확인 필요.

## 8. 시즌 하이라이트 (Seasonal Highlights)

### 테마파크

5월은 비수기(off-peak)로, 대기 시간이 짧고 쾌적한 방문 가능.

| 테마파크 | 운영 시간 | 특징 |
|----------|-----------|------|
| **Warner Bros. Movie World** | 10:00–17:00 | 슈퍼히어로 놀이기구, 퍼레이드 |
| **Sea World** | 10:00–17:00 | 해양 동물 쇼, 놀이기구 |
| **Dreamworld** | 10:00–17:00 | 호주 최대 테마파크, 코알라 체험 |
| **Wet'n'Wild** | 5월에는 수온 관계로 축소 운영 가능 | 워터파크 — 날씨에 따라 방문 고려 |
| **Currumbin Wildlife Sanctuary** | 08:00–16:00 | 코알라·캥거루 체험, 로리킷 먹이주기 |

- 통합 패스(5-Day Multi Park Pass) 이용 시 경제적
- 5월은 학교 방학이 아니므로 한산함

### 힌터랜드 (Hinterland)

5월 하순은 힌터랜드 트레킹의 **최적기** — 서늘하고 건조하며 거머리·모기가 적음.

| 국립공원 | 주요 코스 | 소요 시간 | 하이라이트 |
|----------|-----------|-----------|------------|
| **Springbrook NP** | Purling Brook Falls Circuit | 약 2시간 (4 km) | 폭포, 열대우림, 현수교 |
| **Springbrook NP** | Natural Bridge | 약 1시간 (1 km) | 천연 아치형 동굴, **반딧불이(Glow Worm)** — 야간 투어 인기 |
| **Springbrook NP** | Best of All Lookout | 약 30분 (600 m) | 칼데라 전망, 맑은 날 절경 |
| **Lamington NP (Green Mountains)** | Tree Top Walk | 약 2–3시간 | 남극너도밤나무(Antarctic Beech), 캐노피 산책로 |
| **Lamington NP (Binna Burra)** | Caves Circuit | 약 3–4시간 (5 km) | 동굴, 전망대 |

- **Natural Bridge 반딧불이 투어**: 5월은 어두워지는 시간이 빨라(17시) 야간 투어 접근 용이
- 아침 안개 속 트레킹은 환상적인 경험 — 새소리(Whipbird 등)와 함께하는 숲속 산책
- 1,000년 이상 된 고대 남극너도밤나무 군락은 가을에 특히 신비로운 분위기

## 10. 복장 / 준비물 (What to Wear & Pack)

### 복장 가이드

| 시간대 | 기온 | 추천 복장 |
|--------|------|-----------|
| 낮 (10–15시) | 22–25 °C | 반팔 또는 얇은 긴팔, 면바지/청바지 |
| 아침·저녁 (6–9시, 17–21시) | 13–18 °C | 긴팔 + 가벼운 재킷/가디건 |
| 해변 | 22–24 °C (수온) | 수영복 + 래시가드 (바람 대비) |
| 힌터랜드 트레킹 | 10–18 °C (산지) | 레이어링: 기능성 속옷 + 플리스 + 방풍 재킷 |

### 필수 준비물 체크리스트

- [ ] **레이어링 옷**: 얇은 긴팔, 가디건/후드, 가벼운 바람막이 재킷
- [ ] **긴바지**: 청바지 또는 면바지 (반바지는 낮 전용)
- [ ] **수영복**: 래시가드 포함
- [ ] **자외선 차단제**: SPF 50+ (호주 자외선은 겨울에도 강함)
- [ ] **선글라스 & 모자**: 일조 시간이 길고 자외선 강함
- [ ] **우산/우비**: 간헐적 소나기 대비 (작은 접이식 우산)
- [ ] **트레킹화**: 힌터랜드 방문 시 필수 (미끄러운 열대우림 트레일)
- [ ] **얇은 스카프/목도리**: 저녁 해안 산책 시 유용
- [ ] **방수 가방/파우치**: 해변·보트 투어 시 전자기기 보호
- [ ] **보온 상의**: 고래 관찰 보트 위에서 바람 체감 온도 낮음

> **핵심 키워드는 "레이어링"** — 낮과 밤의 기온차가 11°C 이상이므로, 겹쳐 입고 벗을 수 있는 옷이 최적.

## 종합 요약

| 항목 | 요약 |
|------|------|
| **기온** | 최고 24–25°C / 최저 13–14°C. 한국 5월 중순 낮 + 10월 중순 아침 느낌 |
| **비** | 월 69–100 mm, 짧은 소나기 위주. 야외 활동 방해 적음 |
| **일조** | 하루 ~8.4시간 일조. 일몰 17시 즈음으로 다소 이른 편 |
| **습도/바람** | 64–73% 습도, 풍속 16–21 km/h. 여름 대비 쾌적 |
| **바다** | 수온 22–24°C. 짧은 수영 가능, 서핑 시 2–3mm 웻수트 |
| **고래** | 시즌 막 시작 (5월 30일 투어 오픈). 초기 혹등고래 관찰 가능 |
| **서핑** | 남·남동 스웰 양호. Kirra·Burleigh 최적기. 크라우드 적음 |

| **힌터랜드** | 트레킹 최적기. 반딧불이 투어, 고대 숲 산책 강추 |

| **복장** | 레이어링 핵심. 재킷 + 자외선 차단 필수 |

**골드코스트 5월 하순은 관광 비수기의 장점(한산함, 저렴한 숙박)과 쾌적한 가을 날씨가 결합된 최적의 시기이다.** 고래 관찰 시즌의 시작, 힌터랜드 트레킹의 황금기, 그리고 세계적 수준의 서핑 조건까지 — 다채로운 경험이 가능하다.
