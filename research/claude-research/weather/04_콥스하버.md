# 콥스하버 — 5월 하순 날씨 및 여행 환경

## 1. 기온

| 항목 | 수치 |
|------|------|
| 평균 최고 기온 | 20.6~21 °C |
| 평균 최저 기온 | 13.8~14 °C |
| 평균 기온 (mean) | 약 17 °C |
| 체감 온도 | 낮 쾌적, 아침/저녁 서늘 |

- 5월은 호주 가을(Autumn)의 막바지로, 여름 더위가 완전히 가시고 선선한 날씨
- **한국 비교**: 서울 10월 초~중순과 비슷한 체감. 낮에는 쾌적하고 아침/저녁에 긴소매 필요
- 일교차가 약 7°C로, 레이어링 필수
- 5월 하순으로 갈수록 기온이 서서히 하락하여 6월(겨울) 진입 준비

## 2. 강수

| 항목 | 수치 |
|------|------|
| 월간 강수량 | 약 94 mm (일부 출처 46~94 mm) |
| 비 오는 날 수 | 약 13~15일/월 |
| 일일 비 확률 | 약 16% |

- 5월은 콥스하버의 **건기 전환기**에 해당하며, 여름(1~3월)에 비해 강수량이 크게 줄어듦
- 비가 오더라도 대부분 가벼운 소나기 형태로, 하루 종일 비가 내리는 경우는 드묾
- 우산 또는 경량 레인재킷 정도면 충분

## 3. 일조/일몰

| 항목 | 수치 (5월 하순 기준) |
|------|------|
| 일출 | 약 06:17~06:22 |
| 일몰 | 약 16:55~17:00 |
| 일장 (Day length) | 약 10시간 35분~10시간 40분 |
| 일평균 일조 시간 | 약 7.6~8.2시간 |
| UV 지수 | 평균 최대 5 (중간 수준) |

- 5월 하순은 동지(6월 21일)를 약 한 달 앞둔 시기로, 해가 일찍 짐
- **오후 5시 전후에 해가 지므로** 야외 활동은 오전~오후 초에 집중 권장
- UV 지수 5는 중간(Moderate) 수준이지만, 맑은 날 자외선 차단 필요

## 4. 습도/바람

| 항목 | 수치 |
|------|------|
| 평균 상대습도 | 약 71~73% |
| 오전 9시 습도 | 약 75% 내외 |
| 오후 3시 습도 | 약 60~65% |
| 평균 풍속 | 약 17.5 km/h (10.9 mph) |
| 주요 풍향 | 남서~서풍 (오프쇼어) |

- 습도가 한국 가을보다 다소 높지만, 기온이 낮아 불쾌하지 않음
- 바람은 해안가 특성상 오후에 강해지는 경향 (해풍/시브리즈)
- 5월은 남서풍(서풍)이 우세하여 서핑에 유리한 오프쇼어 바람 조건

## 5. 바다 수온/수영

| 항목 | 수치 |
|------|------|
| 평균 해수 온도 | 약 21 °C |
| 수온 범위 | 20~22 °C |
| 수영 가능 여부 | 가능 (단, 웻수트 권장) |

### 솔리터리 아일랜드 스노클링 (Solitary Islands Marine Park)

- **가시성(시야)**: 최소 15m 이상일 때 투어 출항 (맑은 날 20m+)
- **수온 21°C**는 스노클링에 약간 서늘한 편 — **3mm 웻수트 최소 권장, 추위 민감 시 5mm**
- **Jetty Dive Centre**에서 솔리터리 아일랜드 스노클 반나절 투어 운영
  - 매일 오전 8:30 출항, 11:30~12:00 귀항
  - 2개 스노클링 사이트 방문
  - 웻수트(3/5/7mm), 마스크, 스노클, 핀 전부 포함 — **1인 $75 AUD**
  - 예약 필수, 보증금 $100 (48시간 전 취소 시 환불)
- 동호주 해류(East Australian Current)와 남쪽 찬류가 만나 **열대어 + 온대 해양생물** 동시 관찰 가능
- 그레이 너스 상어(Grey Nurse Shark), 대왕갑오징어(Giant Cuttlefish) 등 5~6월 관찰 확률 상승

## 6. 고래 관찰 (Whale Watching)

| 항목 | 정보 |
|------|------|
| 시즌 | 5월~11월 (혹등고래 북상 이동) |
| 피크 시즌 | 6월~10월 |
| 5월 하순 가능 여부 | **시즌 초입 — 초기 목격 가능** |

### 머튼버드 아일랜드 (Muttonbird Island Nature Reserve)

- 콥스하버 마리나/제티에서 도보 접근 가능 (왕복 약 2km)
- **동쪽 전망대(Eastern Side Lookout)**가 고래/돌고래 관찰 최적 포인트
- 무료, 예약 불필요 (악천후 시 폐쇄 가능)
- 5월 하순은 혹등고래 북상 시작 시기로, 운이 좋으면 목격 가능

### 보트 투어

| 운영사 | 기간 | 특징 |
|--------|------|------|
| **Whale Watch Experience (Pacific Explorer)** | 6월~11월 | 매일 8:30am, 11:00am 출항, 최대 23명 소규모 |
| **Jetty Dive Centre (Mega Wild)** | 6월~10월 | 8~9월 혹등고래 수영 체험 가능 |

- **주의**: 대부분의 보트 투어는 **6월부터** 정식 운항 시작
- 5월 하순(5/23~31)에는 보트 투어가 아직 시작 전일 가능성 높음
- **대안**: 머튼버드 아일랜드 전망대에서 육상 관찰 또는 운영사에 직접 문의하여 조기 출항 가능 여부 확인

## 7. 서핑 조건

| 항목 | 정보 |
|------|------|
| 주요 스웰 방향 | 남동(SE) 그라운드 스웰 |
| 최적 바람 | 남서(SW)~서(W) 오프쇼어 |
| 파고 | 1~2m (5월 평균) |
| 수온 | 21 °C (3mm 웻수트 권장) |
| 베스트 시즌 | 겨울(6~7월) — 5월은 직전 전환기 |

### 주요 서핑 스팟

| 스팟 | 특징 | 난이도 |
|------|------|--------|
| **Park Beach** | 콥스하버 중심 해변, 비치브레이크, 좌우 파도 | 초~중급 |
| **Diggers Beach** | 빅바나나 맞은편, SE 스웰에 좋은 반응 | 중급 |
| **Macauleys Headland** | 포인트 브레이크, 크고 깨끗한 파도 | 중~상급 |
| **North Wall (Harbour)** | 하버 입구, 일관된 파도 | 초~중급 |

- 5월 하순은 겨울 스웰이 본격화되기 시작하는 시기로, **좋은 서핑 조건 기대 가능**
- Park Beach와 Diggers Beach는 모든 조수 상태에서 서핑 가능
- **주의**: Diggers Beach는 비순찰 해변(unpatrolled)이며 이안류(rip current) 주의
- Park Beach 서프 클럽 근처가 상대적으로 안전

## 8. 시즌 하이라이트 / 관광명소

### 빅바나나 (The Big Banana Fun Park)

- 호주 최대 실내 자이언트 슬라이드, 워터파크, 토보건 라이드 (바다 전망)
- 36홀 미니골프, 레이저태그, 방탈출, 범퍼카, 아이스 스케이트, 4D 라이드
- 바나나 농장 투어, 파충류 월드(Reptile World)
- **연중 운영** — 5월에도 모든 시설 이용 가능

### 도리고 국립공원 (Dorrigo National Park)

- 콥스하버에서 차로 약 1시간
- **곤드와나 우림(Gondwana Rainforest)** — 세계유산 등재
- 스카이워크(Skywalk) 전망대: 콥스하버까지 조망 가능, 무료, 항시 개방
- 폭포 트레일: Crystal Shower Falls, Dangar Falls 등
- 5월 가을 단풍과 우림의 조화가 아름다움 — 걷기 좋은 기온
- 새벽 안개 속 우림 탐방이 인상적

### 솔리터리 아일랜드 해양공원 (Solitary Islands Marine Park)

- NSW 최대 해양공원 (약 71,000 헥타르)
- 열대 + 온대 해양 생태계 교차 지점
- 스노클링, 다이빙, 카약, 낚시 등 다양한 수상 활동
- 5~6월: 그레이 너스 상어, 대왕갑오징어, 거북이 관찰 최적기

### 기타 가을 활동

- **콥스 크릭 워크(Coffs Creek Walk)**: 마을 중심부 산책로, 조류 관찰
- **벨링겐(Bellingen)**: 차로 30분, 보헤미안 분위기의 작은 마을, 마켓/카페
- **Sawtell**: 해변 마을, 레스토랑과 부티크, 일요일 파머스 마켓
- **트리탑 어드벤처(Treetops Adventure Park)**: 우림 속 짚라인/로프 코스

## 종합 요약

콥스하버의 5월 하순은 **호주 가을의 끝자락**으로, 한국의 10월 초와 비슷한 쾌적한 날씨를 보인다. 평균 기온 17°C, 최고 21°C/최저 14°C로 야외 활동에 최적이며, 여름철 혼잡을 피한 **비수기의 장점**을 누릴 수 있다.

**핵심 포인트:**

- **기온**: 낮 20~21°C, 밤 13~14°C. 한국 10월 초 체감. 레이어링 필수
- **바다**: 수온 21°C, 스노클링 가능하나 웻수트(3~5mm) 필수. 솔리터리 아일랜드 해양공원에서 열대/온대 해양생물 동시 관찰 가능
- **고래**: 시즌 초입(5~11월). 머튼버드 아일랜드에서 무료 육상 관찰 가능. 보트 투어는 6월부터 본격 운항 — 5월 말은 확인 필요
- **서핑**: 겨울 스웰 시작기, SE 그라운드스웰 + SW 오프쇼어 조건으로 좋은 파도 기대. Park Beach(순찰), Diggers Beach(비순찰) 추천
- **관광**: 빅바나나(실내 액티비티 풍부), 도리고 국립공원(우림+폭포, 차 1시간), 벨링겐 마을 방문 추천
- **이벤트**: 대형 축제보다는 정기 마켓(수/목/일)과 소규모 공연 중심
- **복장**: 낮 반팔~긴팔, 아침/저녁 스웨터+긴바지, 우비 준비. 도리고는 더 따뜻하게

비수기라 숙소 가격이 저렴하고, 해변과 관광지가 한산하여 여유로운 여행이 가능한 시기다.
