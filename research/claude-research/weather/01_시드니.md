# 시드니 — 5월 하순 날씨 및 여행 환경

## 1. 기온

| 항목 | 수치 |
|------|------|
| 평균 최고기온 | 18.3 ~ 19.0 °C |
| 평균 최저기온 | 11.6 ~ 12.1 °C |
| 평균 기온 | 약 15.5 °C |
| 5월 하순 최고기온 범위 | 16 ~ 22 °C |
| 5월 하순 최저기온 범위 | 9 ~ 14 °C |
| 체감 비교(한국) | **서울 10월 중순 ~ 하순** |

- 5월 하순은 시드니의 **가을 끝자락**으로, 월초 대비 기온이 1~2 °C 더 낮아진다. 월초 최고 20.2 °C에서 월말 18.3 °C로 점차 하강.
- 낮에는 쾌적하지만 **아침·저녁은 쌀쌀**하여 얇은 겉옷이 필수.
- 한국 기준으로는 **서울 10월 중순~말** 날씨와 가장 유사하다. 낮에 선선하고 아침저녁 싸늘한 가을 날씨.
- 일교차가 7~8 °C로 상당하므로 레이어링(층층이 입기)이 중요.

## 2. 강수

| 항목 | 수치 |
|------|------|
| 5월 평균 강수량 | 117.4 mm (BOM 장기평균) |
| 강수일수 | 약 12~13일/월 |
| 일평균 강수 확률 | 약 40 % |
| 강수 패턴 | 소나기성, 하루종일 내리는 경우 드묾 |
| UV 지수 | 평균 3 (중간 수준) |

- BOM(호주 기상청) 기준 5월 장기평균 강수량은 **117.4 mm**이지만, 최근 몇 년(2024년 219 mm, 2025년 212 mm)은 평년 대비 높은 강수량을 기록.
- 비는 대부분 **짧은 소나기** 형태로 내리며, 종일 비가 내리는 날은 드물다.
- 접이식 우산이나 가벼운 방수 재킷 휴대를 권장.
- 비가 오지 않는 날은 맑고 쾌적한 가을 하늘이 펼쳐진다.

## 3. 일조/일몰

| 항목 | 수치 |
|------|------|
| 일출 (5/23 기준) | 약 06:38 AEST |
| 일몰 (5/23 기준) | 약 17:02 AEST |
| 일출 (5/31 기준) | 약 06:43 AEST |
| 일몰 (5/31 기준) | 약 16:58 AEST |
| 낮 길이 | 약 10시간 15분 ~ 10시간 24분 |
| 일평균 일조시간 | 약 6.3시간 |

- 남반구 가을이므로 **해가 짧아지는 시기**. 겨울 지점(6월 21일)까지 점점 일조시간이 줄어든다.
- 오후 5시쯤 해가 지므로 **야외 관광은 오전에 집중**하는 것이 효율적.
- Vivid Sydney 라이트 쇼는 일몰 후인 **오후 6시부터 점등**되므로, 해가 진 후 바로 축제를 즐길 수 있다.
- 일평균 실제 일조시간은 6.3시간으로, 흐린 날이 간간이 있다는 의미.

## 4. 습도/바람

| 항목 | 수치 |
|------|------|
| 평균 상대습도 | 57 ~ 69 % |
| 오전 습도 (9시) | 약 68 ~ 72 % |
| 오후 습도 (15시) | 약 52 ~ 57 % |
| 평균 풍속 | 17 ~ 17.5 km/h |
| 주요 풍향 | 서풍~남서풍 |

- 습도는 한국의 10월과 비슷하게 **쾌적한 편**. 여름처럼 끈적이지 않는다.
- 해안가에서는 바람이 체감온도를 2~3 °C 낮출 수 있으므로, **바람막이 재킷**이 유용.
- 평균 풍속 17~17.5 km/h는 산들바람~약간 강한 바람 수준. 보트 투어 시 체감 추위 주의.
- 오전에 습도가 높고 오후에 건조해지는 전형적인 패턴.

## 5. 바다 수온/수영

| 항목 | 수치 |
|------|------|
| 5월 평균 수온 | 20 ~ 21 °C |
| 최저 수온 | 약 19.5 °C |
| 최고 수온 | 약 22.1 °C |
| 수영 쾌적성 | 짧은 수영은 가능, 장시간은 추움 |
| 웻수트 권장 | 2 mm 숏티 또는 3/2 mm 풀수트 |

- 5월 수온은 **20~21 °C**로, 한국 여름 동해안 수온(7~8월)과 비슷하지만 체감상 선선하다.
- 맨몸 수영은 가능하지만 **10분 이상 들어가면 추위**를 느낀다. 특히 이른 아침이나 흐린 날은 체감이 차갑다.
- **웻수트를 입으면 쾌적하게 수영 가능**: 2 mm 숏티(낮 시간) 또는 3/2 mm 풀수트(아침/저녁/바람 부는 날) 권장.
- Bondi Icebergs Pool 등 해수 수영장은 연중무휴로 운영되며, 5월에도 이용 가능.
- 서퍼들은 대부분 3/2 mm 웻수트를 착용한다.

## 6. 고래 관찰

| 항목 | 내용 |
|------|------|
| 시즌 여부 | **시즌 초반** (5월 중순부터 시작) |
| 이동 방향 | 남극 → 북쪽 (번식지로 이동 중) |
| 주요 종 | 혹등고래(Humpback Whale) |
| 성수기 | 6월 말 ~ 7월 초 (수백 마리/일 통과) |
| 보트 투어 운항 | 5월 말부터 일부 운항사 시작 |

- 5월 하순은 고래 시즌의 **바로 시작점**. 혹등고래가 남극에서 북쪽 번식지로 이동하기 시작하는 시기.
- 매년 **30,000마리 이상의 혹등고래**가 시드니 앞바다를 지나간다.
- **보트 투어 운항사**:
  - Captain Cook Cruises: 매일 9:30am, 1:30pm 출발 (Circular Quay), 2.5시간
  - Whale Watching Sydney: Darling Harbour 출발
  - Ocean Extreme: Circular Quay, Manly Wharf 출발 (고속 보트)
  - Go Whale Watching: Darling Harbour 출발, 3시간
- 대부분 운항사가 **고래 미목격 시 무료 재탑승** 보장 (95%+ 목격률).
- **육지 전망대** (무료):
  - Cape Solander (Kamay Botany Bay National Park) — 최고의 전망대, 지붕 있는 관찰 플랫폼
  - North Head (Fairfax Walk) — 1km 산책로, 3개 전망대
  - Barrenjoey Headland (Palm Beach) — 등대에서 360도 조망
- 다만 5월 하순은 시즌 초이므로 목격 수가 적을 수 있음. **피크는 6월 말~7월 초**.

## 7. 서핑 조건

| 항목 | 내용 |
|------|------|
| 시즌 | 가을~겨울이 가장 일관된 파도 시즌 |
| 평균 파고 | 1.0 ~ 1.5 m (때때로 2 m+) |
| 주요 스웰 방향 | 남동(SE), 동(E) |
| 수온 | 20 ~ 21 °C |
| 웻수트 | 3/2 mm 풀수트 권장 |

- 5월은 시드니 서핑의 **베스트 시즌 진입기**. 가을~겨울(3~8월)에 가장 일관된 파도가 들어온다.
- 5~9월에는 대형 **남동(SE) 스웰**이 꾸준히 밀려오며, 이것이 시드니 해변 서핑의 핵심.
- **주요 서핑 스팟**:
  - **Bondi Beach**: 초·중급자 친화적. 남쪽 스웰에 좋은 반응. 서핑 스쿨 다수.
  - **Manly Beach**: 다양한 파도 조건. 노스 스틸(North Steyne) 구간 인기.
  - **Maroubra Beach**: 로컬 서퍼들이 선호. 좀 더 강한 파도.
  - **Cronulla Beach**: 시드니 남부, 비교적 한적한 서핑 스팟.
- 오프쇼어 바람(서풍~북서풍)이 부는 오전에 가장 깨끗한 파도를 만날 수 있다.

- **Vivid Sydney**가 여행 기간과 완벽하게 겹치는 것이 최대 하이라이트.
- Blue Mountains는 가을 단풍이 절정에 달하는 시기. 단풍나무(Wentworth Falls 주변), 자작나무 등이 황금빛으로 물든다.
- 5월은 **비수기 시작**이므로 숙박비가 여름 성수기 대비 20~40% 저렴.
- 시드니 하버 주변 산책, 해안 트레일(Bondi to Coogee Coastal Walk 등)이 쾌적한 기온 덕분에 최적.
- 시드니 피쉬마켓, 차이나타운 야시장 등 미식 투어도 가을이 제격.
