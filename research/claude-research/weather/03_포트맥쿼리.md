# 포트맥쿼리 — 5월 하순 날씨 및 여행 환경

> 호주 뉴사우스웨일스(NSW) 중북부 해안, 시드니에서 북쪽 약 390 km.
> 5월은 호주 가을(Autumn) 마지막 달로, 성수기가 지나 한적하면서도 쾌적한 기후를 즐길 수 있는 시기이다.

---

## 1. 기온 (Temperature)

| 항목 | 수치 |
|------|------|
| 평균 최고 기온 | **20.2 °C** |
| 평균 최저 기온 | **14.6 °C** (새벽~이른 아침) |
| 평균 기온 | **약 17 °C** |
| 체감 온도 | 바람 불면 12~14 °C까지 느껴질 수 있음 |

### 한국 비교
- 서울 5월 평균 최고 24 °C / 최저 13 °C와 비교하면 **낮 기온이 3~4 °C 낮고, 아침 기온은 비슷한 수준**
- 서울의 10월 중순 또는 부산의 11월 초와 체감이 유사
- 큰 일교차(5~6 °C)가 있으므로 레이어링 필수

## 2. 강수 (Rainfall)

| 항목 | 수치 |
|------|------|
| 월간 강수량 | **약 40 mm** |
| 강수 일수 | **약 12일** (가벼운 소나기 포함) |
| 강수 패턴 | 짧은 소나기 형태가 대부분, 하루 종일 비 오는 날은 드묾 |

- 5월은 포트맥쿼리에서 **건기에 해당**하며, 연중 강수량이 가장 적은 달 중 하나
- 여름(1~3월) 월 평균 100~150 mm 대비 현저히 적음
- 갑작스러운 소나기에 대비해 가벼운 방수 재킷 권장

---

## 3. 일조 / 일몰 (Sunrise & Sunset)

| 항목 | 수치 (5월 하순 기준) |
|------|------|
| 일출 | **약 06:20~06:25** (AEST) |
| 일몰 | **약 16:55~17:00** (AEST) |
| 낮 길이 | **약 10시간 30~35분** |
| 일조 시간 (실제 햇빛) | **약 7.4시간/일** (월 평균) |

- 5월 하순은 동지(6월 21일)에 가까워지면서 낮이 짧아지는 시기
- **오후 5시면 어두워지므로** 야외 활동은 오전~오후 이른 시간에 집중 권장
- 일출 방향: 동북동 → 해변에서 일출 감상 가능 (Town Beach, Lighthouse Beach 추천)

---

## 4. 습도 / 바람 (Humidity & Wind)

| 항목 | 수치 |
|------|------|
| 평균 상대 습도 | **약 73%** |
| 평균 풍속 | **약 17 km/h** |
| 주요 풍향 | 서~남서풍 (offshore wind, 서핑에 좋음) |

- 여름 대비 습도가 낮아 **쾌적한 편**
- 한국의 습한 여름(80%+)과 비교하면 체감이 훨씬 건조하고 상쾌
- 해안가 바람이 있으므로 체감 온도 하락에 유의 (윈드브레이커 권장)
- 아침저녁 바람이 강할 수 있으며, 특히 해변가에서는 체감 12~14 °C 가능

---

## 5. 바다 수온 / 수영 (Sea Temperature & Swimming)

| 항목 | 수치 |
|------|------|
| 5월 평균 수온 | **약 21~22 °C** |
| 5월 하순 수온 | **약 20~21 °C** (월말로 갈수록 하락) |
| 수온 범위 (5월 전체) | 19~25 °C |
| 수영 쾌적도 | 짧은 입수 가능, 장시간은 추울 수 있음 |

### 수영 가이드
- **맨몸 수영**: 10~15분 정도 가능하나 체온 하락 빠름
- **래시가드 + 보드숏**: 가벼운 수중 활동에 적합
- **웻수트 권장**: 3/2 mm 풀수트 또는 2 mm 스프링수트
  - 서핑, 스노클링 등 장시간 수중 활동 시 필수
  - 이른 아침이나 바람 부는 날에는 반드시 착용
- 한국 제주도 5월 수온(약 17~18 °C) 대비 **2~3 °C 높아** 상대적으로 나은 편

## 6. 고래 관찰 (Whale Watching)

### 시즌 정보
| 항목 | 내용 |
|------|------|
| 이동 시기 | **5~11월** (남극 → 아열대 북상) |
| 5월 하순 가능성 | **시즌 초반이지만 관찰 가능** |
| 종류 | 혹등고래(Humpback Whale) 중심 |
| 이동 개체수 | 연간 약 16,000마리 이상 통과 |
| 피크 시기 | 6~7월 (하루 300~400마리 관찰 보고) |

### 5월 하순 관찰 전망
- 4월 말부터 하루 2~5마리 목격 보고가 나오기 시작
- **5월 하순은 시즌 초반이지만 관찰 가능성 충분** (매년 빨라지는 추세)
- 6월 대비 개체 수는 적지만, 한적한 관찰 환경이 장점

### Tacking Point Lighthouse (태킹 포인트 등대)
- 포트맥쿼리 해안선에서 **가장 동쪽에 위치한 전망 포인트**
- 남쪽에서 북상하는 고래를 가장 먼저 발견할 수 있는 최적 장소
- 1879년 건립, 호주에서 13번째로 오래된 등대
- **무료 입장**, 접근성 좋은 산책로 설치
- Lighthouse Beach 북단에 위치, 주차 가능

### 보트 투어 (Port Jet Cruise Adventures)
| 옵션 | 세부 사항 |
|------|------|
| Wave Rider | 스피드보트, 가까이 접근 가능 |
| Essence | 쌍동선(카타마란), 3층 전망 데크 + 수중 관찰창 |
| 요금 | **AUD 85~** (성인 기준) |
| 소요 시간 | 1.5~2.5시간 |
| 운항 시기 | 5~11월, 매일 (악천후 제외) |
| 예약 | portjet.com.au / +61 419 620 765 |

> 주말은 혼잡하므로 **사전 예약 권장**

### 육상 관찰 포인트 (무료)
- Tacking Point Lighthouse (최고 추천)
- Sea Acres National Park 해안 전망대
- Harry's Lookout
- North Brother Mountain (국립공원 내)

## 7. 서핑 조건 (Surf Conditions)

### 5월 서핑 개요
| 항목 | 내용 |
|------|------|
| 시즌 평가 | **연중 최고 시기 중 하나** (5~8월 겨울 스웰 시작) |
| 깨끗한 파도 비율 | **62%** (5월 Town Beach 기준) |
| 블로운아웃 비율 | 29% |
| 주요 스웰 방향 | 남동(SE) |
| 이상적 바람 | 서~남서풍 (offshore) |
| 평균 파고 | 1~2.5 m |

### 주요 서핑 스폿

| 스폿 | 레벨 | 특징 |
|------|------|------|
| **Town Beach** | 초~중급 | 남동 스웰 + 서남서 offshore 최적, 2.5 m까지 |
| **Lighthouse Beach** | 중~상급 | 5월이 최적기, 짧고 파워풀한 파도 |
| **Flynn's Beach** | 초급 | 바람 차단 좋음, 초보자에게 안전 |
| **North Breakwall** | 중급 | 일정한 파도, 접근 용이 |
| **Bonny Hills** | 중~상급 | 노출된 포인트 브레이크, 가을/겨울 최적 |

- **웻수트 필수** (3/2 mm 권장)
- 5월은 서남서풍이 많아 offshore 조건이 자주 형성 → 깨끗한 파도

## 8. 시즌 하이라이트 / 관광 명소 (Seasonal Highlights)

### Koala Hospital (코알라 병원)
- **Lord Street 본관**: 2025년 5월부터 대규모 리모델링 공사 중, **2026년 중반 재개장 예정**
  - 여행 시점(5월 하순)에는 재개장 여부 불확실 → 출발 전 반드시 확인 필요
  - 공식 사이트: koalaconservationaustralia.org.au
- **대안: Guulabaa (구울라바) — Wild Koala Breeding Visitor Centre**
  - **주소**: Guulabaa Way, off Burrawan Forest Drive, Lake Innes NSW 2446
  - 코알라 사육장 전망 데크, 인터랙티브 교육 전시, 야생 코알라 번식 프로그램 관람
  - Lord Street 공사 기간 중 방문 가능한 대체 시설
- 야생 코알라 구조 및 재활 전문 시설, 세계적으로 유명

### Sea Acres Rainforest Centre (시에이커스 열대우림)
- NSW 최대 규모의 해안 열대우림 보호구역
- **1.3 km 고가 보드워크** (평탄, 휠체어 접근 가능)
- 양치식물, 덩굴식물, 무화과나무 등 아열대 식생
- 도마뱀(goanna), 다양한 조류 관찰 가능
- 5월은 비교적 한적하여 조용한 산책 가능

### Port Macquarie Coastal Walk (해안 산책로)
- **총 길이**: 약 9 km (Town Beach → Tacking Point Lighthouse)
- **소요 시간**: 3~4시간 (편도)
- **경유지**: Town Beach → Flagstaff Hill → Rocky Beach Lookout → Nobby's Beach → Shelly Beach → Miners Beach → Sea Acres → Lighthouse Beach → Tacking Point Lighthouse
- **5월 장점**: 서늘한 기온으로 걷기에 쾌적, 고래 관찰 병행 가능

### Port Macquarie Camel Safaris (낙타 사파리)
- **장소**: Lighthouse Beach
- **운영**: 예약 불필요 (선착순)
- **소요 시간**: 약 20분
- 해변을 따라 낙타를 타고 이동하는 독특한 경험
- 모자, 선크림 지참 권장

### 기타 추천 활동
- **Hastings River 카약/패들보드**: 잔잔한 강에서 수상 활동
- **North Brother Mountain 전망대**: 야생 동식물 + 360도 파노라마 뷰
- **Ricardoes Tomatoes & Strawberries**: 딸기 따기 체험 (5월에도 가능)
- **Cassegrain Wines**: 와이너리 투어 및 시음
- **Port Macquarie Museum**: 지역 역사 탐방

| **아침/저녁** | 플리스 또는 가벼운 패딩, 윈드브레이커 |
| **해안 산책** | 레이어링 (반팔 + 바람막이), 선글라스, 모자 |
| **서핑/수중 활동** | 3/2 mm 웻수트, 래시가드, 서프부츠 |

| **레스토랑/카페** | 스마트 캐주얼 (청바지 + 니트 정도) |

- [ ] 윈드브레이커 / 방수 재킷 (가벼운 비 대비)

- [ ] 선크림 SPF 50+ (호주 자외선 강함, 5월에도 UV 3~5)
- [ ] 선글라스 (UV 차단)
- [ ] 모자 (챙 넓은 것 권장)
- [ ] 편안한 워킹화 (Coastal Walk 9 km 대비)
- [ ] 수영복 + 래시가드
- [ ] 웻수트 (서핑 계획 시, 현지 렌탈 가능)
- [ ] 쌍안경 (고래 관찰용, 있으면 좋음)
- [ ] 방수 폰 케이스
- [ ] 가벼운 우산 또는 레인코트

## 종합 요약

### 5월 하순 포트맥쿼리 = "가을의 평화로운 해안 도시"

| 평가 항목 | 등급 | 코멘트 |
|-----------|------|--------|
| **날씨 쾌적도** | ★★★★☆ | 서늘하지만 맑은 날 많음. 한국 10월 중순 체감 |
| **야외 활동** | ★★★★★ | 걷기, 하이킹에 최적. 여름 더위·인파 없음 |
| **수영/수상** | ★★★☆☆ | 수온 20~21 °C, 웻수트 없이 장시간 수영 어려움 |
| **서핑** | ★★★★★ | 연중 최고 시기. 깨끗한 파도 62%, offshore wind |
| **고래 관찰** | ★★★★☆ | 시즌 초반이지만 관찰 가능. 6월 대비 한적 |
| **관광 전반** | ★★★★★ | 비수기 요금, 적은 인파, 코알라·열대우림·등대 등 볼거리 풍부 |

### 핵심 포인트
1. **기온**: 낮 20 °C, 밤 15 °C — 한국 가을 날씨와 유사하나 해안 바람으로 체감 온도 낮음
2. **비**: 월 40 mm로 적은 편이나 소나기 가능, 방수 재킷 필수
3. **일몰 빠름**: 오후 5시경 — 야외 활동은 오전~이른 오후 집중
4. **고래**: 5월 하순이면 혹등고래 북상 시작, Tacking Point에서 육안 관찰 가능
5. **서핑**: 연중 최고 시즌 진입, Town Beach·Lighthouse Beach 추천
6. **비수기 장점**: 숙소 저렴, 관광지 한적, 현지인 분위기 만끽
